# 🚀 Shashank Creator Hub

Shashank Creator Hub is a unified dashboard where all web tools are accessible from a single platform.  
Built using **Next.js**, it provides a fast, modern, and scalable experience.

---

## 🌐 Live Platform

All tools are organized in one place for easy access and usage.

---

## ✨ Features

- 🔎 Search functionality for tools
- 🧩 Dynamic tool listing from JSON data
- 🎨 Modern responsive UI (Tailwind CSS)
- ⚡ Fast performance with Next.js
- 📱 Fully responsive (Mobile + Desktop)
- 🗂️ Category-based organization
- 🔗 External tool redirection

## 📌 How It Works

- Tools are stored inside `data/tools.json`
- Each tool contains:
  - id
  - name
  - slug
  - url
  - description
  - icon
  - category
- `ToolCard.tsx` dynamically maps slug to tool URL
- Clicking **Open Tool** opens the respective website in a new tab
